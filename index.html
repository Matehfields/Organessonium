Como funciona

Um app local usando o Discord como cofre de arquivos

O Organessônium não é só uma tela bonita em cima de anexos. Ele mantém um índice, reconstrói arquivos divididos, carrega miniaturas e conversa com o bot para transformar threads em itens navegáveis.

01

Conexão inicial

Você cola apenas o token do bot; o Organessônium encontra sozinho os canais necessários no servidor.

02

Upload em threads

Ao enviar um arquivo, o app cria uma publicação no fórum. Se passar de 10 MB, ele divide em partes e envia em sequência.

03

Índice pesquisável

O canal names guarda nome, tamanho, data, pasta, cor e ID da thread para o app listar tudo rapidamente.

04

Download e remontagem

Na volta, o app baixa os anexos com threads configuráveis, junta as partes e entrega o arquivo final no destino escolhido.

Recursos do app

O essencial de um explorador de arquivos, com extras próprios

A experiência foi pensada para uso frequente: arrastar, selecionar, abrir prévias, editar Markdown, organizar pastas e acompanhar transferências sem precisar lidar diretamente com o Discord.

Arraste para enviar

Solte arquivos na janela ou use o botão de upload. Também dá para enviar pastas inteiras e manter a estrutura organizada.

Seleção como no Explorer

Clique, arraste o mouse e marque vários itens com um retângulo de seleção, parecido com o Windows Explorer.

Mover por arrasto

Depois de selecionar, arraste arquivos para pastas ou para a raiz. O app mostra um fantasma visual do item sendo movido.

Abas de navegação

O app usa abas para alternar entre arquivos e áreas abertas, como o editor Markdown, sem perder o ponto onde você estava.

Prévia de PDFs

PDFs são renderizados em páginas, com carregamento progressivo e camada de texto selecionável quando disponível.

Versão mobile Flutter

A versão Android fica no projeto organessonium-flutter, com interface pensada para toque e distribuição por APK.

Editor Markdown integrado

Arquivos .md podem ser abertos em uma área própria, com modo visual, Markdown puro, toolbar e salvamento automático.

Busca e ordenação

Pesquise por nome e alterne a listagem por nome, tamanho ou data. A interface mantém modos de grade e lista.

Português, inglês e ajustes finos

A interface tem versões em português e inglês, tema claro/escuro, tamanho de cards, prévias, animações e desempenho configuráveis.

Cache local no PC

A versão desktop usa SQLite local para guardar configurações e dados auxiliares sem depender da rede para tudo.

Prévia de imagens

Imagens podem ser abertas no app com visualização rápida, útil para conferir arquivos antes de baixar ou mover.

Ecossistema multiplataforma

Desktop e Android seguem a mesma biblioteca de arquivos, com versões separadas para cada ambiente.

Versão mobile

Interface Android feita em Flutter

A versão mobile fica em organessonium-flutter e segue o mesmo conceito do desktop: buscar, abrir arquivos, enviar itens, acompanhar transferências e navegar pela biblioteca com uma interface própria para toque.
